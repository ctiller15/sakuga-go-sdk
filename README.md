# sakuga-go-sdk

A go SDK for the [Sakugabooru API](https://sakugabooru.com/help/api)

Currently supported api calls:

Posts - List
Tags - List
Tags - Related
Artists - List
Comments - Show
Wiki - List
Wiki - Show
wiki - History
Notes - List
Notes - Search
Notes - History
Users - Search
Forum - List
Pools - List Pools
Pools - List Posts
Favorites - List Users
